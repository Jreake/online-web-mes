# 属性

| 属性         | 说明               |  类型   | 可选值 |        默认值        |
| :----------- | :----------------- | :-----: | :----: | :------------------: |
| tips         | 上传之前的提示     | string  |   --   | 点击或拖拽至此处上传 |
| file         | 上传的文件         | object  |   --   |          --          |
| accept       | 支持上传的文件类型 | string  |   --   |          \*          |
| beforeAvatar | 是否要校验文件类型 | boolean |   --   |        false         |
| format       | 要校验文件类型     | string  |   --   |          ''          |

# 事件

| 属性         | 说明                                 | 回调参数 |
| :----------- | :----------------------------------- | :------: |
| update:file  | 对应 file 属性，文件增加和删除时触发 |   val    |
| beforeUpload | 文件增加时触发                       |   val    |
